# nodejsbookingcare
Project website booking care full stack backend and frontend
Frontend:
+ HTML, SCSS
+ Reactjs framework
+ Bootstrap4 (reactrap)
Backend:
+ Nodejs (expressjs framework)
Database:
+ MySql (Sequelize)
